Configuration Supabase

Pour que les données soient partagées et persistantes entre tous les utilisateurs, connectez votre base Supabase. Opération unique — les paramètres seront mémorisés.

① Créer un compte gratuit Allez sur supabase.com → New Project
② Créer la table SQL Dans l'éditeur SQL de Supabase, exécutez :

create table meetings (id text primary key, day text, time text, location text, host text, "hostCompany" text, guest text, "guestCompany" text, "hostPresence" text default 'unknown', "guestPresence" text default 'unknown', note text default ''); -- Migration si table déjà existante : -- alter table meetings add column "hostPresence" text default 'unknown'; -- alter table meetings add column "guestPresence" text default 'unknown'; -- update meetings set "hostPresence"=presence, "guestPresence"=presence; -- alter table meetings drop column presence;
③ Récupérer vos clés Project Settings → API → URL et anon public key
